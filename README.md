# 发票管理系统

一个基于 Vue 3 + Element Plus 的发票管理系统，实现了用户管理、权限控制、发票管理等功能。

## 功能特点

- 发票管理：上传、识别、导出
- 客户管理：添加、编辑、导出
- 系统日志：操作记录、数据分析
- 权限控制：多角色权限管理
- 数据可视化：ECharts 图表展示

## 技术栈

- Vue 3.3.4
- Element Plus 2.4.4
- ECharts 5.4.3
- Vue Router 4.2.5
- Less 4.2.0

## 开发环境

- Node.js >= 16
- npm >= 8
